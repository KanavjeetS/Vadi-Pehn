# Research Findings

Output of `.agents/skills/research-and-propose.md` and `/research-loop`.
One file per finding: `YYYY-MM-DD-<slug>.md`.

Each file should contain: what changed, why it matters to this specific
project (with a PRD/System Design section reference), a risk
classification (`low-risk` / `needs-review` / `urgent-safety` — see
`research-and-propose.md`), sources with dates, and a concrete next step.

`urgent-safety` findings are also pointed to from the top of
`GUARDRAILS.md` under `## Pending Urgent Research Findings` — check there
first for anything time-sensitive.
